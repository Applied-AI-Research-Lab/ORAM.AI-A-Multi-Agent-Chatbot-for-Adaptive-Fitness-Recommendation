## ✨ Features

- 🤖 **Multi-Agent Architecture** - Intelligent agent orchestration for complex reasoning
- 🧠 **RAG-Based System** - Retrieval-Augmented Generation for accurate responses
- 🔌 **MCP Integration** - Model Context Protocol for enhanced context management
- 💪 **Fitness Recommendations** - Personalized exercise suggestions
- 🎙️ **Voice Integration** - Speech-to-Text and Text-to-Speech capabilities
- 🌐 **Multi-Language Support** - English and Greek language interfaces
- 📊 **Comprehensive Testing** - Full test coverage for chatbot, STT, and TTS
